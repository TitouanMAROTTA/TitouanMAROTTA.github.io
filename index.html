Principe

"CSS Battle" est un défi de design en ligne qui met à l'épreuve les compétences d'un développeur Web en matière de CSS. Avec plus de 140 défis différents, le site cssbattle.dev permet aux développeurs Web d'améliorer leur expérience en CSS de manière ludique.

Chaque défi consiste à recréer un affichage spécifique en utilisant le moins de code possible, ce qui vous oblige à trouver des solutions créatives pour obtenir les résultats souhaités.

Avec des classements mondiaux et des récompenses pour les meilleurs joueurs, CSSBattle est le moyen idéal pour perfectionner ses compétences en CSS.
